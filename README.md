# On the farm

## Access

Go to this repository's Github Pages, or use the link below:

https://ahmed-m-tawfik.github.io/on-the-farm/

Commits to main branch are automatically built and deployed. Hard refresh and game reset may be needed to get full reload.

## Development

Build

```bash
./build.sh
```

Watch

```bash
./build.sh -w
```

Outputs `story.html`, which should open in any browser without requiring a live server.
